Hotel Booking System – Case Study

Implemented Features
- Room search by date and category
- Booking flow
- Loyalty program with points redemption
- Returning customer detection
- Staff dashboard concept

Architecture
- Blazor Server UI
- Service layer for business logic
- In-memory data store for demo

Future Improvements
- Database persistence
- Real-time availability updates
- Payment integration
- Authentication & authorization

AI Usage
AI was used for scaffolding and UI suggestions, but all business logic and validation were reviewed and implemented manually.
